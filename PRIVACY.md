# Privacy Policy

Effective date: 2026-07-06

Kanji Writing Practice is a kanji handwriting practice app.

This app is designed to work locally on the user's device. It does not require account registration and does not send learning data or document contents to an external server.

## Data Collection

This app does not collect personal data.

The app does not use account registration, advertising SDKs, analytics SDKs, tracking technologies, external server communication for learning data, IDFA, or cross-app tracking identifiers.

## Data Stored on the Device

The app stores the following data locally on the user's device:

- kanji practice status
- review state
- mastered state
- selected review interval
- filter settings
- tags created from PDFs
- selected font name
- basic app settings

This data is stored only on the device and is not transmitted by the app.

## PDF Tag Feature

The app can read text from a PDF selected by the user and create tags based on kanji contained in that PDF.

The app does not upload PDF files, PDF text, or extracted kanji to any server.

PDF text extraction is performed locally on the device. OCR is not provided.

## Font Selection

The app may allow the user to select a font installed on the device using the iOS system font picker.

The selected font name may be stored locally on the device so that the app can use the same font later.

The app does not upload font files or font information to any server.

## Speech and Dictionary Features

The app uses iOS system features for speech and dictionary lookup.

The app does not operate its own dictionary server and does not send learning records to an external service.

## Data Sharing

The app does not share user data with third parties.

## Data Deletion

Users can delete locally stored app data by deleting the app from the device.

Some practice states can also be reset within the app settings.

## Children's Privacy

The app does not knowingly collect personal information from children.

The app does not include advertising, tracking, or account registration.

## Changes to This Policy

This privacy policy may be updated when the app's features change.

## Contact

For questions about this privacy policy, please contact the repository owner through GitHub Issues or the support contact listed on the App Store product page.
