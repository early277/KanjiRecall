# 漢字書き取り

漢字書き取り is a kanji handwriting practice app for iOS.

The App Store name is:

情景とパーツで覚える漢字書き取り

The app is designed for cases where a user can read a kanji but cannot easily write it from memory.

## Concept

- Guess the target kanji from an example sentence
- Use scene and kanji parts as memory cues
- Practice writing manually without automatic handwriting judgment
- Create personal practice sets from text-based PDFs

## Features

- Practice writing kanji from reading and context
- Hide the target kanji in the question text
- Manage kanji as untried, review, or mastered
- Select review intervals: tomorrow, one week, one month, or permanent
- Filter by level, tag, and state
- Create kanji tags from text-based PDFs
- Use the iOS dictionary lookup interface
- Use iOS speech synthesis
- Select fonts installed on the device
- Dark mode support
